## Setting up the environment  

### 1. clone the project  
```
git clone https://github.com/hanxuwu/project-HeartDisease.git
```

### 2. set up the virtualenv  

```
> virtualenv .
```

__for windows:__
```
> .\Scripts\activate
```


__for MacOs:__
```
> source ./bin/activate
```

### 3. install requirements  

```
> pip3 install -r requirements.txt
```
